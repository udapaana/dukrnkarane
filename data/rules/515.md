## Insertion of इ before थ् in अद्, कृट्, and हथे

@deva[अद्], @deva[कृट्] and @deva[हथे] admit @deva[इ] necessarily before @deva[थ्].

**@deva[कृट्]**

| | | |
|---|---|---|
| 1. | @deva[आर्] | @deva[आरिव] | @deva[आरिम] |
| 2. | @deva[आरिथ्] | @deva[आरथुः] | @deva[आर] |
| 3. | @deva[आर्] | @deva[आरतुः] | @deva[आरुः] |

@note[type=note]{For @deva[अद्] and @deva[हथे] see p. p. 325 & 324.}

**@deva[भञ्ज्]**

| | | |
|---|---|---|
| 1. | @deva[ममञ्ज] | @deva[ममञ्जिव] | @deva[ममञ्जिम] |
| 2. | @deva[ममञ्जिथ्],@deva[ममङ्क्थ्] | @deva[ममञ्जथुः] | @deva[ममञ्ज] |
| 3. | @deva[ममञ्ज] | @deva[ममञ्जतुः] | @deva[ममञ्जुः] |

**@deva[अज्] 'to go.'**

| | | |
|---|---|---|
| 1. | @deva[विवाय], @deva[विवय्] | @deva[विव्येव], @deva[आजिव] | @deva[विव्येयम्], @deva[आजिम्] |
| 2. | @deva[विवयिथ्], @deva[विवेथ्] | @deva[विव्यथुः] | @deva[विव्य] |
| | @deva[आजिथ्] | | |
| 3. | @deva[विवाय] | @deva[विव्यतुः] | @deva[विव्युः] |
