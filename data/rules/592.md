## Conjugation of Passive Base as Fourth Class

The passive base is conjugated like a root of the 4th class in the Âtm., i. e.[^1]

### Paradigm: @deva[भू] 'to be'

#### Present

| | | | |
|---|---|---|---|
| 1. | @deva[भूये] | @deva[भूयावहे] | @deva[भूयामहे] |
| 2. | @deva[भूयसे] | @deva[भूयेथे] | @deva[भूयध्वे] |
| 3. | @deva[भूयते] | @deva[भूयेते] | @deva[भूयन्ते] |

#### Imperfect

| | | | |
|---|---|---|---|
| 1. | @deva[अभूये] | @deva[अभूयावहि] | @deva[अभूयामहि] |
| 2. | @deva[अभूयथाः] | @deva[अभूयेथाम्] | @deva[अभूयध्वम्] |
| 3. | @deva[अभूयत] | @deva[अभूयेताम्] | @deva[अभूयन्त] |

#### Imperative

| | | | |
|---|---|---|---|
| 1. | @deva[भूये] | @deva[भूयावहै] | @deva[भूयामहै] |
| 2. | @deva[भूयस्व] | @deva[भूयेथाम्] | @deva[भूयध्वम्] |
| 3. | @deva[भूयताम्] | @deva[भूयेताम्] | @deva[भूयन्ताम्] |

#### Potential

| | | | |
|---|---|---|---|
| 1. | @deva[भूयेय] | @deva[भूयेवहि] | @deva[भूयेमहि] |
| 2. | @deva[भूयेथाः] | @deva[भूयेयाथाम्] | @deva[भूयेध्वम्] |
| 3. | @deva[भूयेत] | @deva[भूयेयाताम्] | @deva[भूयेरन्] |

[^1]: It might even be suspected, that the occasional assumption of an Intransitive signification and a Parasmaipada inflexion by a passive verb, was the cause which gave rise to a 4th class of Primitive verbs as distinct from the Passive. Instances of passively formed of passive verbs taking Parasmaipada terminations, and some passive verbs (e. g. @[jāyate] 'is born,' tr. @[jan]; @[pūryate], 'is filled,' tr. @[pṛ]; and @[upyate] 'is heated,' tr. @[tap]) are regarded by native grammarians as Âtmanepada verbs of cl. 4. Again, many roots appear in class 4 as Intransitive verbs, which also appear in some one of the other nine as Transitive. For example, @[yuj], 'to join' when used in a Transitive sense, is conjugated either in cl. 7, or in the Causal; when in an Intransitive, in cl. 4. So also @[puṣ], 'to nourish,' @[kṣubh], 'to agitate;' @[kliś] 'to vex'; @[ślāgh], 'to accomplish,' &c. — Monier Williams.
