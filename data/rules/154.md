@deva[एक] when a numeral is declined in the sing. and @deva[द्वि] in the dual only, @deva[द्वि]. is considered as @deva[द्वि], and they are both declined like @deva[सर्व] in all the three genders; e g.

| N. | @deva[एकः] | @deva[एका] | @deva[द्वौ] | @deva[द्वे] | |
| V. | @deva[एक] | @deva[एके] | ,, | ,, | |
| A. | @deva[एकम्] | @deva[एकाम्] | @deva[द्वौ] | @deva[द्वे] | |
| I. | @deva[एकेन] | @deva[एकया] | @deva[द्वाभ्याम्] | The rest like the mas. | |
| D. | @deva[एकस्मै] | @deva[एकस्यै] | ,, | | |
| Ab. | @deva[एकस्मात्] | @deva[एकस्याः] | ,, | | |

| L. | @deva[एकस्मिन्] | @deva[एकस्याम्] | ,, | | |

@deva[एक] neu. @deva[एकम्] N. A. The rest like the mas.

@deva[एक] when it does not mean 'one' is declined in the dual and plural also.
