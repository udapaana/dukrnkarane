## Base of अधि+इ (to study): अधीज्

The base of @deva[इ] with @deva[अधि] 'to study' is @deva[अधीज्].

| | | |
|---|---|---|
| 1. | @deva[अधिजगे] | @deva[अधिजगिवहे] | @deva[अधिजगिमहे] |
| 2. | @deva[अधिजगिषे] | @deva[अधिजगाथे] | @deva[अधिजगिध्वे] |
| 3. | @deva[अधिजगे] | @deva[अधिजगाते] | @deva[अधिजगिरे] |
