The following roots (some forming their P. P. P. irregularly) reject @deva[इ] in the limited sense attached to each:—

| | | | |
|---|---|---|---|
| @deva[चुष्]—@deva[चुष्टः] | 'a churning handle.' | @deva[विसर्]—@deva[विसृटः] | 'a note.' |
| @deva[स्मरु]—@deva[स्मृतः] | 'the mind.' | @deva[कृष्]—@deva[फृष्टः] | 'a decoction easily prepared.' (@deva[अनासासमासो: कदाचिदुक्तः] Sid. Kau.) |
| @deva[स्थुप्]—@deva[स्थातः] | 'darkness.' | | |
| @deva[लग्]—@deva[लग्नः] | 'attached.' | @deva[चाहृ]—@deva[चाटः] | 'much.' |
| @deva[स्वैत्]—@deva[स्वातः] | 'indistinct.' | | |

In their usual senses they have @deva[हृषितः], @deva[स्पर्शितः], @deva[क्षमितः], @deva[म्लेश्छितः], @deva[विरञ्जितः], @deva[फणितः] and @deva[वाह्यितः].
