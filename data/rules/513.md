## Irregular Perfect of भू

@deva[भू] forms its base irregularly as @deva[बभूव]—

**Parasmaipada:**

| | | |
|---|---|---|
| 1. | @deva[बभूव] | @deva[बभूविव] | @deva[बभूविम] |
| 2. | @deva[बभूविथ्] | @deva[बभूवथुः] | @deva[बभूव] |
| 3. | @deva[बभूव] | @deva[बभूवतुः] | @deva[बभूवुः] |

**Ātmanepada:**

| | | |
|---|---|---|
| 1. | @deva[बभूवे] | @deva[बभूविवहे] | @deva[बभूविमहे] |
| 2. | @deva[बभूविषे] | @deva[बभूवाथे] | @deva[बभूविध्वे] |
| 3. | @deva[बभूवे] | @deva[बभूवाते] | @deva[बभूविरे] |
