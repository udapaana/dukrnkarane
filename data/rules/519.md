## Substitution of स्था and कृञा for स्था in Perfect

@deva[स्था] and @deva[कृञा] are optionally substituted for @deva[स्था] in the Perfect, and necessarily in the other Non-conjugational forms.

@deva[स्था] and @deva[कृञा] take both the Padas.

**Parasmaipada:**

| | | |
|---|---|---|
| 1. | @deva[आचस्थौ] | @deva[आचस्थिविव] | @deva[आचस्थियम्] |
| | @deva[आचक्रौ] | @deva[आचक्रिषव] | @deva[आचक्रिषाम्] |
| 2. | @deva[आचस्थ्याथ्], @deva[आचस्थिय्थ्] | @deva[आचस्थ्यथुः] | @deva[आचस्थ्य] |
| | @deva[आचक्राथ्], @deva[आचक्रिश्थ्] | @deva[आचक्रथुः] | @deva[आचक्रा] |
| 3. | @deva[आचस्थौ] | @deva[आचस्थ्यतुः] | @deva[आचस्थ्युः] |
| | @deva[आचक्रौ] | @deva[आचक्रथुः] | @deva[आचक्रयुः] |

**Ātmanepada:**

| | | |
|---|---|---|

| | @deva[आचस्थे] | @deva[आचस्थियवहे] | @deva[आचस्थियमहे] |
| | @deva[आचक्रे] | @deva[आचक्रिवहे] | @deva[आचक्रिमहे] |
| 2. | @deva[आचस्थिषे] | @deva[आचस्थाथे] | @deva[आचस्थिध्वे] |
| | @deva[आचस्थ्यषे] | @deva[आचस्थ्याथे] | @deva[आचस्थ्यध्वे]-@deva[ढ्] |
