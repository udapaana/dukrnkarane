## Anomalous Causal Forms

The following roots form their Causal base anomalously:

| Root | Meaning | Causal Form | Causal Meaning |
|------|---------|-------------|----------------|
| @deva[इ] | 'to go' | @deva[गमयति] | |
| | with @deva[अधि] 'to remember' | @deva[अधिगमयति] | 'to study' |
| | with @deva[प्रति] | @deva[प्रत्यापयति] | |
| @deva[ध्वन] or @deva[ध्वान] | 'to sound,' &c. | @deva[ध्वनयति] | 'he causes to sound' |
| @deva[क्षमाव्] | 'to tremble' | @deva[क्षमावयति] | 'he causes to tremble' |
| @deva[मुह्] | 'to conceal' | @deva[मोहयति] | 'he causes to conceal' |
| @deva[चि] 5 cl. | 'to collect' | @deva[चापयति-ते], @deva[चायपयाति-ते] | 'to collect' |
| @deva[चि] 10 cl. | | @deva[चयपयति-ते], @deva[चयपयाति-ते] | |
| @deva[जागृ] | 'to awake' | @deva[जागरयति] | 'he rouses' |
| @deva[दुष्] | 'to sin, to be unchaste' | @deva[दुष्यति-ते] | 'he causes to sin' |
| | | @deva[दूषयति-ते], @deva[दोषयति-ते] | 'he corrupts or makes depraved' |
| @deva[भू] | 'to shake' | @deva[भूनयति-ते] | 'he causes to shake' |
| @deva[प्री] | 'to please' | @deva[प्रीणयति] | 'he causes to please' |
| @deva[भी] | 'to fear' | @deva[भाययति-ते] with @deva[भापयाते], @deva[भीषयते] | 'he frightens' / 'he inspires fear' |
| @deva[भञ्ज्] | 'to fry' | @deva[भर्जयति-ते], @deva[भ्रञ्जयति-ते] | 'he causes to fry' |
| @deva[मार्ज्] | 'to wipe' | @deva[मार्जयति] | |
| @deva[रञ्ज्] | 'to dye' | @deva[रञ्जयति] | 'he dyes or paints; he propitiates or satisfies'[^1] |
| @deva[रुह्] | 'to grow' | @deva[रोहयति-ते], @deva[रोपयति-ते] | 'he plants or causes to grow' |
| @deva[ला] | 'to take' | @deva[लालयति-ते], @deva[लिलायति-ते]; and @deva[लम्भयति], @deva[लापयति] | 'melting an unctuous substance' |
| @deva[ली] | 'to embrace, to adhere' | | |
| @deva[वा] | 'to blow' | @deva[वातयति] | 'he causes to blow or move' |
| | | @deva[वाञ्जयति] | 'he shakes' |
| @deva[स्मि] | 'to smile' | @deva[स्मेमायति] | 'he causes a smile by, or astonishes or frightens by' |
| | | @deva[स्मेत्मापते] | 'he astonishes' |
| @deva[व्ये] | | @deva[व्यापयति], @deva[व्यासयति] | 'he causes to conceive' |
| | | @deva[व्यायपति-ते] | 'in other senses' |
| @deva[शद्] | 'to fall' | @deva[शातयति] | 'he causes to fall, cuts down' |
| | | @deva[शासयति] | 'he causes to go' |
| @deva[सिध्] | 'to be accomplished' | @deva[सापयति] | 'he accomplishes or prepares' |
| | | @deva[सिधयति] | 'he makes perfect, &c. with reference to sacred rites or things only; as @deva[स्वपति तापसं वा]: &c.' |
| @deva[स्फाय्] | 'to swell' | @deva[स्फावयति] | 'he causes to swell' |
| @deva[स्फुर्] | 'to tremble; to shine forth' | @deva[स्फोरयति], @deva[स्फुरयति] | 'he causes to tremble or shine' |
| @deva[हन्] | 'to strike or kill' | @deva[घातयति] | 'he causes to strike, &c.' |
