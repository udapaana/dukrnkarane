The rules given at @ref[394], @ref[395], @ref[459], @ref[502] and @ref[587] apply to the roots also in the gerund in @deva[य]:—

| | |  | | |
|---|---|---|---|---|
| @deva[विद्] with @deva[प्र]—@deva[प्रवीद्य] | | @deva[अञ्च्] with @deva[प्र]—@deva[प्राञ्च्य] |
| @deva[कृ] with @deva[अव]—@deva[अवकीर्य] | | @deva[वस्] with @deva[प्र]—@deva[प्रोष्य] |
| @deva[युज्] with @deva[आ]—@deva[आयुज्य] | | @deva[शङ्ख्] with @deva[वि]—@deva[विशङ्ख्य] |
| @deva[बन्ध्] with @deva[नि]—@deva[निबध्य] | | @deva[हा] with @deva[आ]—@deva[आहाय] |
| @deva[हि], @deva[नी], @deva[मा] and | | @deva[दा] with @deva[उद्]—@deva[उद्दाय] |
| @deva[गृ][^1] with @deva[अनु]—@deva[अनुगाय] | | @deva[धा] with @deva[वि]—@deva[विधाय] or @deva[विद्धाय] |
| @deva[रुष्] with @deva[परि]—@deva[परिरुष्य] | | &c. | &c. | &c. |
| @deva[वै] with @deva[आ]—@deva[आवाय] |
