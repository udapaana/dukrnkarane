## Optional First Aorist for प्रा, घे, शो, सो, and छो

The roots @deva[प्रा], @deva[घे], @deva[शो], @deva[सो], and @deva[छो] belong to this Variety optionally. They optionally take the Sixth Form. @deva[घे] takes the Third Form also.

Paradigms.

**@deva[स्था] P.**

| | | |
|---|---|---|
| 1. | @deva[अस्थाम्] | @deva[अस्थाव] | @deva[अस्थाम] |
| 2. | @deva[अस्थाः] | @deva[अस्थातम्] | @deva[अस्थात] |
| 3. | @deva[अस्थात्] | @deva[अस्थाताम्] | @deva[अस्थुः] |

**@deva[शो] P.**

| | | |
|---|---|---|
| 1. | @deva[अशाम्] | @deva[अशाव] | @deva[अशाम] |
| 2. | @deva[अशाः] | @deva[अशातम्] | @deva[अशात] |
| 3. | @deva[अशात्] | @deva[अशाताम्] | @deva[अशुः] |
