## Perfect Base of ऊर्ण्: ऊर्णुव्

@deva[ऊर्ण्] forms its base as @deva[ऊर्णुव्]. Its vowel is optionally not guṇated before a strong termination when it takes @deva[इ].[^1]

**Parasmaipada:**

| | | |
|---|---|---|
| 1. | @deva[ऊर्णनाव्], @deva[ऊर्णुव्] | @deva[ऊर्णुवविव] | @deva[ऊर्णुवविम्] |
| 2. | @deva[ऊर्णुव्विथ्] @deva[ऊर्णुवक्थ्] | @deva[ऊर्णुवथुः] | @deva[ऊर्णुव] |
| 3. | @deva[ऊर्णनाव्] | @deva[ऊर्णुववतुः] | @deva[ऊर्णुववुः] |

**Ātmanepada:**

| | | |
|---|---|---|
| 1. | @deva[ऊर्णुवे] | @deva[ऊर्णुविवहे] | @deva[ऊर्णुविमहे] |
| 2. | @deva[ऊर्णुविषे] | @deva[ऊर्णुवाथे] | @deva[ऊर्णुविध्वे]-@deva[ढ्] |
| 3. | @deva[ऊर्णुवे] | @deva[ऊर्णुवाते] | @deva[ऊर्णुविरे] |
