Tabular representation of the declension of @deva[दिश्].

## FEMININE.

Singular.

| | | | |
|---|---|---|---|
| Nom. | @deva[दिक्] or @deva[दिङ्] | Voc. | @deva[दिक्] |
| Acc. | @deva[दिशम्] | Ins. | @deva[दिशा] |
| Dat. | @deva[दिशे] | Abl. | @deva[दिशः] |
| Gen. | @deva[दिशः] | Loc. | @deva[दिशि] or @deva[दिक्] |

Dual.

| | | | |
|---|---|---|---|
| Nom., voc., and acc. | @deva[दिशौ] | Instr., dat., and abl. | @deva[दिग्भ्याम्] or @deva[दिङ्भ्याम्] or @deva[दिक्भ्याम्] |
| Gen. and loc. | @deva[दिशोः] |

Plural.

| | | | |
|---|---|---|---|
| Nom. and voc. | @deva[दिशः] | Acc. | @deva[दिशः] or @deva[दिगः] |
| Instr. | @deva[दिग्भिः] or @deva[दिङ्भिः] or @deva[दिक्भिः] | Dat. and abl. | @deva[दिग्भ्यः] or @deva[दिङ्भ्यः] or @deva[दिक्भ्यः] |
| Gen. | @deva[दिशाम्] | Loc. | @deva[दिक्षु] or @deva[दिक्सु] |

## NEUTER.

(@deva[उदक्])

Singular.

| | | | |
|---|---|---|---|
| Nom., voc., and acc. | @deva[उदक्] | Instr. | @deva[उदका] |
| Dat. | @deva[उदके] | Abl. | @deva[उदकः] |
| Gen. | @deva[उदकः] | Loc. | @deva[उदकि] or @deva[उदगि] or @deva[उदचि] |

Dual.

| | | | |
|---|---|---|---|
| Nom., voc., and acc. | @deva[उदची] or @deva[उदगी] or @deva[उदचा] | Instr., dat., and abl. | @deva[उदग्भ्याम्] or @deva[उदङ्भ्याम्] |
| Gen. and loc. | @deva[उदचोः] |

Plural.

| | | | |
|---|---|---|---|
| Nom., voc., and acc. | @deva[उदक्] or @deva[उदञ् च] or @deva[उदगि] | Instr. | @deva[उदग्भिः] or @deva[उदङ्भिः] |
| Dat. and abl. | @deva[उदग्भ्यः] or @deva[उदङ्भ्यः] | Gen. | @deva[उदचाम्] |
| Loc. | @deva[उदक्षु] |

**Note.**—The student should in this way make the declension tables of the words @deva[प्राञ्च्] (eastern), @deva[प्रत्यञ्च्] (western), and @deva[अवाञ्च्] (southern), which, like @deva[उदञ्च्] (northern), are derived from the root @deva[अञ्च्] by the affix @deva[क्विप्] or @deva[क्विन्] (according to the genders mas. and neu.)
