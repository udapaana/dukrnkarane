## Reduplication before आम् in भी, ह्री, घू, and हु

When @deva[आम्] is added to the roots @deva[भी], @deva[ह्री], @deva[घू] and @deva[हु] they are first reduplicated as in the third conjugation; e. g.—

**@deva[भी] 2. P. 'to fear.'**

| | | |
|---|---|---|
| 1. | @deva[चिभाय्], @deva[चिभय्] | @deva[चिभियिव] | @deva[चिभियिम्] |
| | @deva[चिभयामाचकार्]-@deva[चकर्], &c. | @deva[चिभयामाचक्रव्], &c. | @deva[चिभयामाचकम्], &c. |
| 2. | @deva[चिभयिथ्], @deva[चिभेथ्] | @deva[चिभ्यथुः] | @deva[चिभ्य] |
| | @deva[चिभयामाचकर्थ्], &c. | @deva[चिभयामाचकृथुः], &c. | @deva[चिभयामाचकृ], &c. |
| 3. | @deva[चिभाय्] | @deva[चिभ्यतुः] | @deva[चिभ्युः] |
| | @deva[चिभयामाचकार्], &c. | @deva[चिभयामाचकृतुः], &c. | @deva[चिभयामाचकुः], &c. |

**@deva[ह्री] 3 P. 'to be ashamed.'**

| | | |
|---|---|---|
| 1. | @deva[जिह्राय], @deva[जिह्रय्] | @deva[जिह्रियिव] | @deva[जिह्रियिम्] |
| | @deva[जिह्रयामाचकार्]-@deva[चकर्], &c. | @deva[जिह्रयामाचक्रव्], | @deva[जिह्रयामाचकम्], &c. |
| 2. | @deva[जिह्रयिथ्]-@deva[जिह्रेथ्] | @deva[जिह्रियथुः] | @deva[जिह्रिय] |
| | @deva[जिह्रयामाचकर्थ्] &c. | @deva[जिह्रयामाचकृथुः], &c. | @deva[जिह्रयामाचकृ], &c. |
| 3. | @deva[जिह्राय] | @deva[जिह्रियतुः] | @deva[जिह्रियुः] |
| | @deva[जिह्रयामाचकार्], &c. | @deva[जिह्रयामाचकृतुः], &c. | @deva[जिह्रयामाचकुः], &c. |

**Additional forms:**

@deva[घू]—@deva[बुभार्], @deva[बुभर्], @deva[बुभरामाचकार्]-@deva[बुभूवामास्] 1 sing.

@deva[हु]—@deva[जुहाव्], @deva[जुहवामाचकार्]-@deva[बुभूवामास] 1 sing. &c.
