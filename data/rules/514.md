## Changes to जि, हि, and क्षि after Reduplication

@deva[जि] and @deva[हि] are respectively changed to @deva[गि] and @deva[घि], and @deva[क्षि] optionally to @deva[ङि], after the reduplicative syllable in the Perf. and the Desiderative.

**@deva[जि]**

| | | |
|---|---|---|
| 1. | @deva[जिगाय] | @deva[जिगियिव] | @deva[जिगियिम] |
| | @deva[जिगय] | | |
| 2. | @deva[जिगेथ] | @deva[जिग्यथुः] | @deva[जिग्य] |
| | @deva[जिगयिथ्] | | |
| 3. | @deva[जिगाय] | @deva[जिग्यतुः] | @deva[जिग्युः] |

**@deva[हि]**

| | | |
|---|---|---|
| 1. | @deva[जिघाय] | @deva[जिघियिव] | @deva[जिघियिम] |
| | @deva[जिघय] | | |
| 2. | @deva[जिघेथ] | @deva[जिघ्यथुः] | @deva[जिघ्य] |
| | @deva[जिघयिथ्] | | |
| 3. | @deva[जिघाय] | @deva[जिघ्यतुः] | @deva[जिघ्युः] |

**@deva[क्षि]**

| | | |
|---|---|---|
| 1. | @deva[चिक्षाय]-@deva[चिक्षाय्] | @deva[चिकियव] | @deva[चिकियम] |
| | @deva[चिचाय]-@deva[चिच्य्] | @deva[चिच्यव] | @deva[चिच्यम] |
| 2. | @deva[चिकियिथ्]-@deva[चिकेथ्] | @deva[चिकयथुः] | @deva[चिकय] |
| | @deva[चिचयिथ्]-@deva[चिचेथ्] | @deva[चिच्यथुः] | @deva[चिच्य] |
| 3. | @deva[चिक्षाय]-@deva[चिक्राय] | @deva[चिकयतुः]-@deva[चिच्यतुः] | @deva[चिकयुः]-@deva[चिच्युः] |
