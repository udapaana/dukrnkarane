## Formation of Present Ātmanepada Participle

The participle of the Present Ātmanepada is formed by adding @deva[आन] to the root which undergoes the same changes before it, as before the termination @deva[अते] or @deva[अन्ते] of the 3rd plural present.

### आन → मान Substitution

@deva[आन] is changed to @deva[मान], in the case of roots of the 1st, 4th, 6th and 10th classes (i.e. in derived verbs; see @ref[482]).

If the base ends in @deva[अ], before which the @deva[अ] of the base remains unchanged.

### Examples

| Root (Class) | Participle | Meaning |
|--------------|------------|---------|
| @deva[एध्] (1) | @deva[एधमान] | growing |
| @deva[वन्द्] (1) | @deva[वन्दमान] | saluting |
| @deva[शये] (2) | @deva[शयान] | lying down |
| @deva[द्विष्] (2) | @deva[द्विषाण] | hating |
| @deva[दूव] with @deva[आ] (2) | @deva[आप्रुवाण] | killing |
| @deva[धृ] (3) | @deva[दधान] | holding |
| @deva[हु] (3) | @deva[जुह्वाण] | sacrificing |
| @deva[दू] (4) | @deva[दृष्यमाण] | desirous of extracting |
| @deva[तुद्] (6) | @deva[तुद्यमान] | giving pain |
| @deva[रुज्] (7) | @deva[रुज्यान] | obstructing |
| @deva[कृ] (8) | @deva[कुर्वमाण] | doing |
| @deva[तन्] (8) | @deva[तन्वमान] | stretching |
| @deva[क्री] (9) | @deva[क्रीणान] | buying |
| @deva[चुर्] (10) | @deva[चोरयमाण] | stealing, etc. |

### Derived Forms

@deva[बुध्] or @deva[चुस्] (1) cau.: @deva[बोध्यमान]—@deva[बोधयमाण] 'causing to know'

Desiderative: @deva[बुबोधिषमाते]—@deva[बुबूधयमाण] 'being desirous to know', etc.
