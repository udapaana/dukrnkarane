## Paradigms and Irregular Frequentatives

### Paradigms: Parasmaipada Frequentatives

The Third persons of the Parasmaipada Frequentative of @deva[कृ] 'to do', @deva[पृच्] 'to fill', and @deva[कृष्] 'to draw furrows'.

All those tenses of this Intensive in which the weak form of the base is used (vide @ref[485], @ref[486]) substitute @deva[चकृ] for @deva[कृ] 'to do'; @deva[पूर्य] 'to fill up', and @deva[कृष्य] 'to draw furrows' respectively.

If in any weak form of an Intensive the root has a final consonant, @deva[र्] is inserted between the Guṇa of the root and the termination (see @ref[640]).

| | @deva[अकोकृषत्, अकोकृषताम्, अकोकृषन्] | |
| **Imperative** |
| | @deva[चेकृतु, चेकृताम्, चेकरतु] | @deva[पेपृचतु, पेपृचताम्, पेपरतु] |
| | @deva[कोकृषतु, कोकृषताम्, कोकृषन्तु] | (or @deva[कोकृतु, कोकृताम्, कोकरन्तु]) |

| | @deva[चेकृयात्] | @deva[पेपृचताम्] |

The following roots form their Frequentatives irregularly:

| Root | Ātm. Freq. | Par. Freq. |
|------|------------|------------|
| @deva[ऊर्णु] 'to cover' | @deva[ऊर्णोनूयते] | — |
| @deva[कु] 1 cl. 'to sound' | @deva[कोकूयते] | — |
| @deva[खन्] 'to dig' | @deva[चंखन्यते] or @deva[चंखन्यते] or @deva[चाखायते] | @deva[चंखनीति] @deva[चंद्रुन्ति], etc. |
| @deva[गृ] 'to swallow' | @deva[जेगिल्यते] | @deva[जागर्ति] |
