As a general rule, all roots ending in a vowel, and such as take the @deva[इ] optionally before any termination, and Aniṭ roots ending in consonants generally, do not take @deva[इ]; the rules of Sandhi given in the foregoing chapters that are applicable in particular cases ought to be observed,—

| Roots. | P. P. Ptc | | Roots. | P. P. Ptc. |
|---|---|---|---|---|
| @deva[पा] | @deva[पात] protected | | @deva[त्यज] | @deva[त्यक्त] abandoned. |
| @deva[क्षि] | @deva[क्षित] resorted to | | @deva[भुञ्ज] | @deva[भुग्ध] fried. |
| @deva[नी] | @deva[नीत] carried. | | @deva[यज्] | @deva[इष्ट] sacrificed |
| @deva[श्रु] | @deva[श्रुत] heard. | | @deva[दुश्] | @deva[दुष्ट] awakened. |
| @deva[भू] | @deva[भूत] become. | | @deva[विध्] | @deva[विद्ध] pierced. |
| @deva[कृ] | @deva[कृत] done. | | @deva[स्तृ] | @deva[स्तृत] spread. |
| @deva[ऊर्णु] | @deva[ऊर्णुत] covered. | | @deva[लभ] | @deva[लब्ध] obtained. |
| @deva[वै] | @deva[ऊत] woven | | @deva[बन्धु] | @deva[बद्ध] bound. |
| @deva[व्ये] | @deva[वीत] covered | | @deva[दृष] | @deva[दृष्ट] seen. |
| @deva[हे][^1] | @deva[हूत] called &c., | | @deva[हिद] | @deva[छिद्ध] cut out &c. |

| Roots. | P. P. Ptc. | | Roots. | P. P. Ptc. |
|---|---|---|---|---|
| @deva[वच्] | @deva[उक्त] spoken | | @deva[दंश्] | @deva[दष्ट] bitten |
| @deva[गुह्] | @deva[गूढ] concealed | | @deva[द्विष्] | @deva[द्विष्ट] hated, &c. |
| @deva[मृज्] | @deva[मृष्ट] wiped off, &c. | | @deva[शास्] | @deva[शिष्ट] governed |
| @deva[सिध्] | @deva[सिद्ध] accomplished, &c. | | @deva[दह्] | @deva[दग्ध] burnt |
| @deva[तृप्] | @deva[तृप्त] satisfied | | @deva[वह्] | @deva[ऊढ] borne |
| @deva[नश्] | @deva[नष्ट] perished, &c. | | @deva[सह्] | @deva[सोढ] endured |
| @deva[वृध्] | @deva[वृद्ध] grown | | @deva[क्षुध्] | @deva[क्षुस्त] destroyed |
| @deva[भू] | @deva[भूत] happened, completed, &c. | | @deva[लिह्] | @deva[लीढ] licked |
| @deva[शक्] | @deva[शक्त] able | | @deva[मुह्] | @deva[मुग्ध] or @deva[मूढ] faint |
| @deva[सिच्] | @deva[सिक्त] sprinkled | | @deva[नह्] | @deva[नद्ध] bound |
| @deva[पृच्छ] | @deva[पृष्ट] asked | | @deva[क्षिप्] | @deva[क्षिप्त] dropped |

**Exceptions:—**

(a) @deva[ज्ञा], @deva[जाग्], @deva[स्था] and @deva[आस्य] take @deva[इ]; the final vowel of the first two takes Guṇa substitute, and the last two drop their final; @deva[ज्ञायित], @deva[जागरित]; @deva[स्थित], @deva[आस्यदित].

(b) @deva[पत्] takes @deva[इ] though it admits of @deva[इ] optionally in Desiderative; @deva[पतित].

(c) The Anit roots @deva[वस्] and @deva[श्लिष्] admit @deva[इ] before @deva[त्]; @deva[उषित], @deva[शुषित].

[^1]: @deva[हे] substitutes @deva[ह] for its @deva[इ].
