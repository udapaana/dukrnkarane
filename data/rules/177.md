The following is an alphabetical list of the irregular comparatives and superlatives that are commonly met with:-

| Positive | Comparative | Superlative |
|---|---|---|
| @deva[अन्तिक] 'near'[^1] | @deva[नेदियस्] | @deva[नेदिष्ठ] |
| @deva[अल्प] 'little'[^2] | @deva[अल्पीयस्] | @deva[अल्पिष्ठ] |
| | @deva[कनीयस्] | @deva[कनिष्ठ] |
| @deva[बृह] 'wide'[^3] | @deva[वरीयस्] | @deva[वरिष्ठ] |
| @deva[क्षिप्र] 'quick'[^4] | @deva[क्षेपीयस्] | @deva[क्षेपिष्ठ] |
| @deva[क्षुद्र] 'mean,' 'small' | @deva[क्षोदीयस्] | @deva[क्षोदिष्ठ] |
| @deva[गुरु] 'heavy' | @deva[गरीयस्] | @deva[गरिष्ठ] |
| @deva[तृप] 'restless or anxious' } | @deva[त्रपीयस्] | @deva[त्रपिष्ठ] |
|     'satisfied'         } | | |
| @deva[दीर्घ] 'long' | @deva[द्राघीयस्] | @deva[द्राघिष्ठ] |
| @deva[दूर] 'distant' | @deva[दवीयस्] | @deva[दविष्ठ] |
| @deva[प्रशस्य] 'praiseworthy'[^5] | @deva[श्रेयस्] or @deva[ज्यायस्] | @deva[श्रेष्ठ] or @deva[ज्येष्ठ] |
| @deva[प्रिय] 'dear' | @deva[प्रेयस्] | @deva[प्रेष्ठ] |
| @deva[बहु] 'much'[^6] | @deva[भूयस्] | @deva[भूयिष्ठ] |
| @deva[बहुल] ,, | @deva[बहीयस्] | @deva[बहिष्ठ] |
| @deva[बाढ] 'firm, well' | @deva[साढीयस्] | @deva[साढिष्ठ] |
| @deva[युवन्] 'young' | @deva[कनीयस्], @deva[कनीयस्] | @deva[यविष्ठ], @deva[कनिष्ठ] |
| @deva[विपुल] 'much' | @deva[ज्यायस्] | @deva[ज्येष्ठ] |
| @deva[वृद्ध] 'old' | @deva[वर्षीयस्], @deva[ज्यायस्] | @deva[वर्षिष्ठ], @deva[ज्येष्ठ] |
| @deva[शठारक] 'great lovely' | @deva[श्रेयीयस्] | @deva[श्रेयिष्ठ] |
| @deva[स्थिर] 'steady' | @deva[स्थेयस्] | @deva[स्थेष्ठ] |
| @deva[स्थूल] 'big, large' | @deva[स्थवीयस्] | @deva[स्थविष्ठ] |
| @deva[स्फिर] 'much' | @deva[स्फायस्] | @deva[स्फायिष्ठ] |
| @deva[ह्रस्व] 'short' | @deva[ह्रसीयस्] | @deva[ह्रसिष्ठ] |

[^1]: @deva[अन्तिकबाह्योर्नेडसाधौ ।] Pāṇ. V. 3. 36.

[^2]: @deva[युवाल्पयोः कनन्यतरस्याम् ।] Pāṇ. 3. 54.

[^3]: @deva[क्षियाक्षिपरिक्षिपेष्वहल्युडब्दुलश्रुदृप्रथीघच्छन्दारकाणाकाणां प्रथमस्य तितक्षुरिचबृप्राघिच्छन्दः ।] Pāṇ. VI. 4. 157. @deva[क्ष], @deva[स्थ], @deva[स्क], @deva[बृ] &c. are substituted for @deva[क्षिप्], @deva[स्थिर], @deva[स्क्रिप्], &c. respectively.

[^4]: @deva[स्थूलदृढुयोडाभीमभूयःश्लाघ्याणां शणादियुत् पूर्वस्य च गुणः ।] Pāṇ. VI 4. 156. The words @deva[स्थूल] &c. drop their final portion beginning with @deva[यण्] (@deva[य], @deva[र], @deva[ल्] or @deva[व्]) and the preceding vowel takes guṇa.

[^5]: @deva[प्रशस्यस्य श्रः । ज्य च । उच्चस्य च] Pāṇ. V. 3. 60-62. @deva[बह्वल्पोपौ च बहोः । क्षुद्रस्य क्षोद्र च ।] Pāṇ. VI. 4. 158-159.

[^6]: See note X p. 112.
