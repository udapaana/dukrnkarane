When a root is compounded with one or more prepositions or words prefixed to verbs like prepositions, the indeclinable participle is formed by affixing @deva[य] immediately to it; the @deva[य] is changed to @deva[त्य] after a short radical vowel (even when it combines with the final vowel of a preposition to a long vowel); e. g.

| | |  | | |
|---|---|---|---|---|
| @deva[दा] with @deva[आ]—@deva[आदाय] | | @deva[इ] with @deva[प्र]—@deva[प्रेत्य] |
| @deva[विद्] with @deva[वि]—@deva[विज्ञाय] | | @deva[कृ] with @deva[सम्]—@deva[संस्कृत्य] |
| @deva[जि] with @deva[परा]—@deva[पराजित्य] | | @deva[कृ] with @deva[वि]—@deva[विकृत्य] |
| @deva[नी] with @deva[वि]—@deva[विनीय] | | @deva[वद्] with @deva[नि]—@deva[निवद्य] |
| @deva[भू] with @deva[अनु]—@deva[अनुभूय] | | @deva[पद्] with @deva[उत्]—@deva[उत्पद्य] |
| @deva[इ] with @deva[अभि]—@deva[अभीत्य] | | &c. | &c. | &c. |
