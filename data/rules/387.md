**Terminations:**—

**Present**

| | Par. | | | Ātm. | | |
|---|---|---|---|---|---|---|
| | S. | D. | P. | S. | D. | P. |
| 1. | @deva[मि] | @deva[वस्] | @deva[मस्] | @deva[ए] | @deva[वहे] | @deva[महे] |
| 2. | @deva[सि] | @deva[थस्] | @deva[थ] | @deva[से] | @deva[एथे] | @deva[ध्वे] |
| 3. | @deva[ति] | @deva[तस्] | @deva[अन्ति] | @deva[ते] | @deva[एते] | @deva[अन्ते] |

**Imperfect**

| | Par. | | | Ātm. | | |
|---|---|---|---|---|---|---|
| | S. | D. | P. | S. | D. | P. |
| 1. | @deva[अम्] | @deva[व] | @deva[म] | @deva[ए] | @deva[वहि] | @deva[महि] |
| 2. | @deva[स्] | @deva[तम्] | @deva[त] | @deva[थास्] | @deva[एथाम्] | @deva[ध्वम्] |
| 3. | @deva[त्] | @deva[ताम्] | @deva[अन्] | @deva[त] | @deva[एताम्] | @deva[अन्त] |

**Imperative**

| | Par. | | | Ātm. | | |
|---|---|---|---|---|---|---|
| | S. | D. | P. | S. | D. | P. |
| 1. | @deva[आनि] | @deva[आव] | @deva[आम] | @deva[ऐ] | @deva[आवहै] | @deva[आमहै] |
| 2. | —* | @deva[तम्] | @deva[त] | @deva[स्व] | @deva[एथाम्] | @deva[ध्वम्] |
| 3. | @deva[तु]* | @deva[ताम्] | @deva[अन्तु] | @deva[ताम्] | @deva[एताम्] | @deva[अन्ताम्] |

* @deva[तात्] is optionally added in the 2nd and 3rd person singular when the Imperative has a Benedictive sense.

**Potential**

| | Par. | | | Ātm. | | |
|---|---|---|---|---|---|---|
| | S. | D. | P. | S. | D. | P. |
| 1. | @deva[ईयम्] | @deva[ईव] | @deva[ईम] | @deva[ईय] | @deva[ईवहि] | @deva[ईमहि] |
| 2. | @deva[ईः] | @deva[ईतम्] | @deva[ईत] | @deva[ईथाः] | @deva[ईयाथाम्] | @deva[ईध्वम्] |
| 3. | @deva[ईत्] | @deva[ईताम्] | @deva[ईयुः] | @deva[ईत] | @deva[ईयाताम्] | @deva[ईरन्] |

**Note:**—Terminations beginning with vowels may be called Vowel terminations; those beginning with consonants, consonantal terminations.
