## Optional Substitution of घस् for अद् in Perfect

@deva[घस्] is optionally substituted for @deva[अद्] in the Perfect.

| | | |
|---|---|---|
| 1. | @deva[आद] | @deva[आविव] | @deva[आविम] |
| | @deva[जघास], @deva[जघस] | @deva[जक्षिव] | @deva[जक्षिम] |
| 2. | @deva[आदिथ्] | @deva[आवथुः] | @deva[आद] |
| | @deva[जघसिथ्] | @deva[जक्षथुः] | @deva[जक्ष] |
| 3. | @deva[आद] | @deva[आवतुः] | @deva[आवुः] |
| | @deva[जघास] | @deva[जक्षतुः] | @deva[जक्षुः] |
