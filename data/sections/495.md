## § 495. *Special rules about the admission of the intermediate @deva[इ] before the terminations @deva[व], @deva[म], @deva[व], @deva[से], @deva[वहे],

§ 495.  *Special rules about the admission  of the intermediate @deva[इ] before the terminations @deva[व], @deva[म], @deva[व], @deva[से], @deva[वहे], @deva[महे]; see (§457 ).

    ( a )  All Set roots admit it, whether set  or  anit, except @deva[इ], @deva[य], @deva[य], @deva[हु], @deva[व]; i, @deva[य], @deva[इ], @deva[य]; @deva[ग्रु], permit @deva[इ इ] or @deva[अ]; But @deva[इ] with @deva[समु], and @deva[इ] admit  it only as @deva[तस्करीय], @deva[वस्तीय].

    ( b )  Anit roots ending in short  @deva[अ],  except  @deva[अ].  reject it before @deva[य]; as @deva[तस्तय्] from @deva[स्य]; but @deva[आरिय] from @deva[इ].

    ( c ) Anit roots with a final vowel or with  a  penultimate @deva[अ] admit it optionally before @deva[य].
