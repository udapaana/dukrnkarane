§ 55. The case-terminations technically called @deva[विभक्ति] are as follows:—

Nom. @deva[स्] or @deva[सु] for the Singular ( @deva[एकवचन] ), @deva[औ] for the Dual ( @deva[द्विवचन] ), @deva[जस्] for the Plural (@deva[बहुवचन]); so @deva[अम्] for the Acc. Sing., @deva[औट्]: ( with final @deva[ट्] dropped ) for the Dual and @deva[शस्] for the Plural; @deva[ट्या] for the Inst. Sing., @deva[भ्याम्] for the Dual, and @deva[भिस्] for the Plural; @deva[डे] (with the initial @deva[ड] dropped) for the Dat. Sing., @deva[भ्याम्] for the dual and @deva[भ्यस्] for the Plural; @deva[डसि] ( = @deva[सि] ) for the Abl. Sing., @deva[भ्याम्] for the Dual, and @deva[भ्यस्] for the Pl; @deva[डस्] ( @deva[ङ] or @deva[स्] ) for the Gen. Sing., @deva[ओस्] for the Dual and @deva[आम्] for the plural; @deva[डि] (=@deva[इ] ) for the loc. Sing., @deva[ओस्] for the Dual, and @deva[सुप्] (=@deva[सु] ) for the Plural.
