## § 19. Similar Vowel Combination

If a simple vowel, short or long, be followed by a similar vowel, short or long, the substitute for them both is the similar long vowel* e.g. @deva[देव + अरि: = देवारि:] @[deva + ariḥ = devāriḥ]; @deva[यज्ञ + आसीत् = यज्ञासीत्] @[yajña + āsīt = yajñāsīt]; @deva[पद्म + अभवत् = पद्माभवत्] @[padma + abhavat = padmābhavat]; @deva[विद्या + आतुर: = विद्यातुर:] @[vidyā + āturaḥ = vidyāturaḥ] eager to gain knowledge; @deva[इति + ईश = इतीश] @[iti + īśa = itīśa]; @deva[अपि + ईश्वते = अपीश्वते] @[api + īśvate = apīśvate]; @deva[श्री + ईश: = श्रीश:] @[śrī + īśaḥ = śrīśaḥ]; @deva[भानु + उदय = भानूदय:] @[bhānu + udaya = bhānūdayaḥ]; @deva[साधु + ऊरु: = साधूरु:] @[sādhu + ūruḥ = sādhūruḥ]; @deva[चमू + ऊर्जी: = चमूर्जी:] @[camū + ūrjīḥ = camūrjīḥ] the strength of the army; @deva[कवि + ऋजु: = कवीर्जु:] @[kavi + ṛjuḥ = kavīrjuḥ], @deva[छृ + ऋकार: = छॄकार:] @[chṛ + ṛkāraḥ = chṝkāraḥ]; @deva[होतृ + टृकार: = होतॄकार:] @[hotṛ + ṭṛkāraḥ = hotṝkāraḥ] the ṭkāra pronounced by the sacrificer (since there is no long @deva[टृ] @[ṭṛ] long @deva[ऋ] @[ṛ] is substituted for both).

### (a) Optional Substitution for ऋ/टृ

If @deva[ऋ] @[ṛ] or @deva[टृ] @[ṭṛ] be followed by a short @deva[ऋ] @[ṛ] or @deva[टृ] @[ṭṛ], short @deva[ऋ] @[ṛ] or @deva[टृ] @[ṭṛ] is optionally substituted for both;[^1] @deva[होतृ + ऋकार: = होतृकार:] @[hotṛ + ṛkāraḥ = hotṛkāraḥ], and also @deva[होतर्कार:] @[hotarkāraḥ] (see § 23, subsection b); so three forms altogether @deva[होतृकार:, होतकार:] @[hotṛkāra:, hotakāraḥ] and @deva[होतर्कार:] @[hotarkāraḥ]; @deva[होतृ + टृकार: = होतृटृकार:] @[hotṛ + ṭṛkāraḥ = hotṛṭṛkāraḥ] and also @deva[होतृट्कार:] @[hotṛṭkāraḥ].

\* @deva[अकः सवर्णे दीर्घः] Pāṇ. VI. 1. 101.
