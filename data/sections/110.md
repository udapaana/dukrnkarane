## § 110. ( 1 ) Bases ending in @deva[न्] drop that letter along

§ 110. ( 1 ) Bases ending in @deva[न्] drop that letter along
with the nom. sing. affix @deva[स्], if there be a short vowel before the
@deva[न्] ; if that vowel be long, the dropping is optional ( Pan. VIII.
2. 7, 23 ); thus @deva[राजन्] nom. sing. @deva[राजा]; @deva[वरिमन्] nom. sing. @deva[वरिमा] ;
@deva[धनिन्] nom. sing. @deva[धनी] ; @deva[मेघिन्] nom. sing. @deva[मेघी] ; @deva[पथिन्] nom. sing. @deva[पन्थाः] or
